# Gulce

Bu depo (repository), Python öğrenim süreci ve denemeleri için oluşturulmuştur.

## Dosyalar

### 1. `gulce.py`
Basit bir Python betiğidir. "Hello World" yazısını ekrana yazdırır.
Çalıştırmak için:
```bash
python3 gulce.py
```

### 2. `Colab'e_hoş_geldiniz_.ipynb`
Google Colab tarafından sağlanan, Jupyter Notebook formatında bir giriş dosyasıdır. İçerisinde Python ile veri görselleştirme ve temel hesaplama örnekleri bulunur.
